# Slearn
Python-C package for performing bayesian automated inference on data, based on probabilistic graphical models.

This package have the following dependencies:

 - decorator 
 - itertools (built-in)
 - matplotlib ( built-in
 - networkx (version  > 2.6)
 - joblib 
 - os
 - pandas
 - pypickle
 - scipy
 - statsmodels
 - tqdm
 - wget
